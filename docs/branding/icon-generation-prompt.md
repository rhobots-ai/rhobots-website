# Rhobots Brand Icon Set - Generation Prompt

## Brand Overview
Rhobots is an enterprise AI solutions company focused on production-grade, secure AI deployment. The brand combines futuristic 3D aesthetics with professional reliability. The visual identity emphasizes geometric precision, metallic materials, and vibrant gradient color schemes.

## Primary Brand Icon Requirements

### Style Guidelines
- **Aesthetic**: Modern, geometric, with subtle 3D depth
- **Complexity**: Clean and professional, not overly complex
- **Line weight**: Bold, confident strokes (consistent 2-3px at base size)
- **Shape language**: Geometric forms - octahedrons, icosahedrons, faceted surfaces
- **Effects**: Subtle metallic sheen, optional glow/emission effects
- **Versatility**: Must work at 16px minimum size, scale up to 512px

### Color Specifications

**Primary Brand Color:**
- Rhobots Blue: #088DFF, #3b82f6 (RGB: 59, 130, 246)
- Accent: Cyan/Teal #14b8a6, #60a5fa

**Theme Variants:**
- Light mode: Full color with #088DFF primary
- Dark mode: Light/white outlines with blue glow (#3b82f6 at 60% opacity)
- Monochrome: Single color versions in gray-900 or white

### Main Logo Icon Options

**Option 1: "Rho Symbol" (ρ)**
- Stylized Greek letter rho (ρ) representing "Rhobots"
- Geometric, angular interpretation with straight edges and precise curves
- Can incorporate orbital ring around the rho for "robotic" feel
- Gradient fill: Linear from #088DFF (top-left) to #06b6d4 (bottom-right)

**Option 2: "Orbital Hub"**
- Central geometric core (octahedron or dodecahedron)
- 3-5 small nodes orbiting around the center
- Connecting lines/paths between nodes suggesting data flow
- Primary shape in Rhobots Blue, orbital nodes in cyan accent

**Option 3: "R-Bot Mark"**
- Stylized "R" letterform combined with robotic/mechanical element
- Angular, geometric construction with faceted edges
- Can include subtle circuit board pattern or node connections
- Solid Rhobots Blue with optional gradient overlay

**Option 4: "Geometric Brain/Network"**
- Low-poly geometric brain or neural network representation
- Interconnected nodes forming an abstract head/intelligence shape
- Emphasizes AI/machine learning aspect
- Blue-to-purple gradient (#3b82f6 to #8b5cf6)

## Product-Specific Solution Icons

### 1. Studio Icon (Custom AI Training)
**Colors**: Purple-to-Pink gradient (#9333ea to #ec4899)
**Concepts**:
- Geometric octahedron with faceted surface
- Learning curve or ascending graph integrated into shape
- Books/knowledge combined with neural network
- Training weights/dials with AI elements
**Visual style**: Bold, confident, represents customization and power

### 2. Extract Icon (Document Intelligence)
**Colors**: Emerald-to-Teal gradient (#10b981 to #14b8a6)
**Concepts**:
- Icosahedron with document layers emerging
- Magnifying glass over structured data grid
- Document with highlighted/extracted fields
- Funnel shape transforming unstructured to structured
**Visual style**: Precise, clean, represents accuracy and extraction

### 3. Sage Icon (Natural Language BI)
**Colors**: Blue-to-Indigo gradient (#3b82f6 to #6366f1)
**Concepts**:
- Dodecahedron with chat bubble or query symbol
- Database cylinder with speech/text input
- Question mark transforming into data visualization
- Brain connected to database/analytics
**Visual style**: Intelligent, approachable, represents natural interaction

### 4. Pulse Icon (Call Center AI)
**Colors**: Pink-to-Red gradient (#ec4899 to #ef4444)
**Concepts**:
- Torus shape with waveform running through it
- Headset combined with audio waveform/heartbeat
- Circular pulse waves radiating from center
- Human silhouette with voice/sound indicators
**Visual style**: Dynamic, human-centered, represents communication

### 5. Operator Icon (Browser Automation)
**Colors**: Orange-to-Amber gradient (#f97316 to #f59e0b)
**Concepts**:
- Torus knot with automation gear/workflow elements
- Browser window with robotic arm/cursor
- Circular workflow arrows with play button
- Geometric robot icon with browser tabs
**Visual style**: Mechanical, automated, represents workflow efficiency

## Technical Specifications

### Export Requirements
- **Formats**: SVG (primary), PNG (16px, 32px, 64px, 128px, 256px, 512px)
- **Color modes**: Full color, monochrome (white), monochrome (black)
- **Variants**: With text (lockup), icon only, favicon (square), app icon
- **Artboards**: 512×512px for icon-only, 1200×400px for horizontal lockup

### Design System Integration
- All icons should maintain visual consistency with geometric 3D theme
- Support gradient overlays (use gradient-compatible fills)
- Maintain 1:1 or 3:2 aspect ratios
- Include padding/safe area (10% margin minimum)
- Ensure contrast ratios meet WCAG AA standards (4.5:1 minimum)

### File Naming Convention
```
rhobots-icon-[variant]-[size].[format]
rhobots-logo-[variant]-[size].[format]
rhobots-solution-[product]-[size].[format]

Examples:
- rhobots-icon-primary-512.svg
- rhobots-logo-horizontal-dark.svg
- rhobots-solution-studio-128.png
```

## Visual Effects (Optional Enhancements)

For premium/hero contexts:
- **Glow effect**: Outer glow 8-16px, color-matched to icon primary, 40-60% opacity
- **Metallic gradient**: Add subtle white-to-transparent gradient overlay at 10-20% opacity for sheen
- **Particle accents**: Small dots/particles around icon edges (3-5 particles max)
- **Depth shadow**: Soft drop shadow 0px 8px 24px rgba(0,0,0,0.15)
